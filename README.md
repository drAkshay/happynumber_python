# happynumber_python
To check whether a number is happy or not :) 

    Happy numbers
 
 
    1 => Sum ( digits ^ 2 )  = 1
    1 => 1^2 = 1
 
    10 => 1^2 + 0^2 = 1
 
    68 => 6^2 + 8^2 = 100 => 1
 
    Unhappy numbers
    0 => 0^2 = 0
 
    4, 16, 37, 58, 89, 145, 42, 20, 4
    2, 4, ..., 4
 
    IsHappyNumber (number) => true if number is happy
    
    num: int; input number
    returns: True if number is happy, False if number is unhappy
